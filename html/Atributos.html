Atributos Globales

Estos atributos pueden usarse en cualquier elemento HTML.

  1. id → único del elemento.
  2. class → una o varias clases CSS.
  3. style → estilos en línea.
  4. title → un texto al pasar el cursor.
  5. lang → el idioma del contenido.
  6. dir → ón del texto (ltr, rtl, auto).
  7. hidden → el elemento.
  8. tabindex → el orden de tabulación.
  9. draggable → arrastrar el elemento (true o false).
  10. contenteditable → el contenido editable.
  11. spellcheck → /desactiva la corrección ortográfica.
  12. accesskey → una tecla de acceso rápido.
  13. data-* → personalizados (por ejemplo: data-user="123").
  14. translate → si el texto debe ser traducido (yes o no).
  15. role → el rol para accesibilidad (ARIA).

Atributos del elemento "form"

Se usan para configurar el comportamiento del formulario.

  1. action → a la que se envían los datos.
  2. method → étodo HTTP (get, post).
  3. enctype → de codificación de los datos (application/x-www-form-urlencoded, multipart/form-data, text/plain).
  4. target → ónde se abrirá la respuesta (_self, _blank, _parent, _top).
  5. name → del formulario.
  6. autocomplete → o desactiva el autocompletado (on, off).
  7. novalidate → la validación al enviar el formulario.
  8. accept-charset → de caracteres permitidas (ej. UTF-8).
  9. rel → ón del formulario con el documento (HTML5).

Atributos del elemento "Input

Controlan cómo funciona y se muestra cada campo.

  1. type → de campo (text, email, password, number, etc.).
  2. name → uiNombre del campo (clave al enviar datos).
  3. value → por defecto.
  4. placeholder → que aparece cuando el campo está vacío.
  5. required → obligatorio.
  6. readonly → No se puede modificar.
  7. disabled → Deshabilita el campo.
  8. maxlength → Número máximo de caracteres.
  9. min / max → Valores mínimo y máximo (numéricos o de fecha).
  10. step → Incremento permitido (para números o fechas).
  11. pattern → ón regular para validar.
  12. checked → una opción por defecto (checkbox o radio).
  13. multiple → Permite seleccionar varios valores (para file o email).
  14. autofocus → el foco automático al cargar la página.
  15. autocomplete → o desactiva el autocompletado.
  16. list → una lista de sugerencias (datalist>).
  17. form → el campo a un formulario específico.
  18. minlength / maxlength → mínima y máxima del texto.
  19. src → URL de imagen (para type="image").
  20. alt → Texto alternativo (para type="image").
  21. accept → de archivos permitidos (para type="file").
