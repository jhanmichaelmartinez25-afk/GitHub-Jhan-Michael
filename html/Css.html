Acerca de

Que es CSS?

CSS (siglas en inglés de Cascading Style Sheets, que se traduce como Hojas de Estilo en Cascada) es un lenguaje de hojas de estilo fundamental para el desarrollo web. Se utiliza para controlar la apariencia visual y el diseño de los documentos escritos en lenguajes de marcado como HTML o XML.


Como usar CSS con HTML?

CSS externo

Este método implica tener un archivo CSS separado (.css) que se vincula al documento HTML. Es ideal para aplicar estilos consistentes a múltiples páginas de un sitio web.

Pasos:

  1. Crea un archivo CSS: tu editor de texto, crea un nuevo archivo y guárdalo con la extensión .css (ej. estilos.css). Este archivo solo debe contener reglas CSS, sin etiquetas HTML.


  2. Vincula el archivo a HTML: tu archivo HTML (ej. index.html), dentro de la sección head, añade la etiqueta link.

CSS interno

Este método se usa cuando una sola página HTML tiene estilos únicos que no se aplican a otras páginas. Los estilos se definen en una etiqueta style dentro de la sección head del documento html.



CSS en linea

Este método aplica un estilo único a un solo elemento HTML específico utilizando el atributo style directamente en la etiqueta de apertura del elemento. No se recomienda para proyectos grandes debido a que dificulta el mantenimiento.
