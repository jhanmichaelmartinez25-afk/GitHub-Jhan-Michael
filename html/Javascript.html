Que es Java Script?

JavaScript (comúnmente abreviado como JS) es un lenguaje de programación fundamental para el desarrollo web. Es uno de los tres pilares de la web moderna, junto con HTML y CSS, y se utiliza principalmente para crear contenido dinámico e interactivo en las páginas web.

Como utilizar JavaScript con HTML?

JS externo

Este es el método más limpio y preferido para la mayoría de los proyectos. Mantienes tu lógica JavaScript en un archivo separado (.js), lo cual facilita la organización y el mantenimiento.

  1. Crea un archivo JS: un archivo nuevo y guárdalo con la extensión .js (ej. script.js).


  2. Vincula el archivo a HTML: En tu archivo HTML, usa la etiqueta script con el atributo src (source) que apunta a la ruta de tu archivo JS.


JS interno

Este método se usa cuando el script es muy corto o es específico de una sola página y no se necesita en ningún otro lugar. El código JavaScript se coloca dentro de la etiqueta script directamente en el documento HTML.



JS en linea

Este método se utiliza para ejecutar pequeñas porciones de JavaScript en respuesta a un evento específico (como un clic de ratón) directamente dentro de un atributo HTML. No se recomienda para lógica compleja, solo para disparadores simples.
